Notas previas:

Favor traer su laptop (los que tengan) y tener instalado Python (cualquier versión entre 2.6 y 2.7, no instalar python 3) y las siguientes librerías de python: numpy, scipy, matplotlib y pyfits.


Para chequear si tienen una determinada librería en un

terminal correr el comando:


python -c "import numpy"


Si corre sin dar mensajes es que SÍ tienen instalada la librería

en cuestión. Si no está instalada, aparecerá un mensaje como: 

    File "<string>", line 1, in <module>

    ImportError: No module named numpy


Opciones de instalación:


Para instalar las librerías de python pueden usar alguna de las siguientes

opciones:


- Bajar e instalar la distribucion Enthought Python (ya contiene python, numpy, scipy y matplotlib). Pyfits

  se baja independientemente (busquenla en google).


-En un terminal hacer:  

    sudo apt-get install python  (para instalar python)

    sudo apt-get install python-numpy (para instalar numpy por ejemplo)


- Si ya tienen python, pueden correr en un terminal:


     easy_install numpy
